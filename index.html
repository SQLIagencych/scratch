Scratch

Welcome to Scratch v0.3.0

Scratch is a functional playground for the front-end architecture of a static-site web project.

Fork it on Github

Scratch is built by and for SQLI Agency Switzerland, but if you find it useful, feel free to clone it, fork it or report issues. Scratch is not really intended to serve as a boilerplate or starting point for a project, though you could certainly use it as one if you'd like. However, a secondary goal of Scratch is to serve as a potential end-point for a custom yeoman-generator which we will develop later.
